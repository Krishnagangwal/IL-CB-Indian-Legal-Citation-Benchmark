# IL-CB-Indian-Legal-Citation-Benchmark

**Status: Under Construction**

A large-scale benchmark for Legal Citation Prediction (LCP) from Supreme Court of India Reports (SCI Reports), 2016–2026.

## Pipeline
- PDF extraction using pdfplumber
- Citation pattern matching (SCC, SCR, AIR formats)
- JSONL output with case_id, context, citations

## Current Scale
- Documents processed: 47 (Jan–Feb 2026)
- Documents ready: 318
- Target: ~7,870 cases (2016–2026)
- Target: ~55,000 citations (2016-2026)
